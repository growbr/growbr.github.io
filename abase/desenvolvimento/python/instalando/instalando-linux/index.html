Iniciando

Olá amigos eu sou o GreenMind e hoje vamos criar a infra necessário para desenvolver para Python no sistema operacional Linux , estou usando para fazer esse tutorial o sistema Debian 8 (Jessie).

Verificando versão

Por padrão o Python já vem instalado no Linux , você pode ver com o simples comando a versão.

$ python --version

Procurando IDLE

No Linux só é necessário instalar o IDLE , mais é muito fácil no linux! No meu caso estou usando o Debian 8 (Jessie) e com o simples comando.

$ apt-cache search idle

Como retorno vai vim muita coisa , mais preste atenção nessas três linhas ...

idle - IDE for Python using Tkinter (default version)
idle-python2.7 - IDE for Python (v2.7) using Tkinter
idle3 - IDE for Python using Tkinter (default version)
idle-python3.4 - IDE for Python (v3.4) using Tkinter

Qual devo instalar ?

Agora você precisa escolher qual a versão do IDLE você deseja instalar , um exemplo se você está com o Python 2.7 você precisa instalar o idle-python2.7 , caso você esteja usando a versão ou o Python 3.4 você precisa instalar a versão idle-python3.4.

Um exemplo melhor é o seguinte , entre na linha de comando e de o seguinte comando para verificar a versão do seu Python.

$ python --version

Caso apareça Python 2.7.9 então você precisa instalar o idle-python2.7 , com o seguinte comando você pode fazer isso.

$ sudo apt-get install idle-python2.7

OBS: Nesse caso por motivos de segurança estamos utilizando o SUDO , nesse link você pode conhecer o SUDO e com certeza não vai se arrepender.

OK! Mais se estiver usando o python3 ? Simples!

$ sudo apt-get install idle-python3.4

Simples não ?

Desenvolvimento ->

IDLE (Using Python-2.7) IDLE (Using Python-3.4)

Vamos testar ?

Assim que estiver feito os passos acima e estiver com o IDLE aberto de o seguinte comando.

print ('Alô Mundo')

Agora já estamos prontos para começar a programar nessa linguagem incrível.
