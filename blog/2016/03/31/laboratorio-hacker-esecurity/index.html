Laboratório Hacker eSecurity

A um tempo a eSecurity mantem um projeto que se chama Laboratório Hacker , o projeto nada mais era que um site com diversas falhas para estudo , com o tempo infelizmente o site saiu do AR porem a VM do projeto foi compartilhada para download.

Entre as falhas temos na VM

  • Wordpress
  • Joomla
  • Escalação de previlegios
  • Brute Force
  • Command Execution
  • CSRF
  • File Inclusion
  • SQLInjection
  • SQLInjection (bind)
  • Upload
  • XSS Reflected
  • XSS Stored

O que precisamos ?

Você pode usar a Maquina Virtual em qualquer sistema operacional , só é necessário algum software de virtualização. Recomendamos fortemente o uso do VirtualBox que está disponível para diversos sistemas Linux , Windows , OS X e até Solares.

Caso tenha dificudades aqui [ABase configurando VirtualBox] , caso use outro programa de virtualização e acha interessante mande uma mensagem que em breve compartilhamos.
